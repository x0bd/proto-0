Working on something small: a research project exploring machine emotion through algorithmic faces.

It's called DOT, an SVG-based avatar that responds to your voice, cursor position, and gestures—mapping emotions (joy, sadness, surprise, anger, curiosity) to subtle geometric changes in real-time.

The aesthetic is minimal but expressive. Clean surfaces, maximum feeling. Every interaction feels intentional.

Right now it's just an experiment. Maybe it becomes an npm SDK one day (`npm install @dot/faces`). Maybe it doesn't. But I've hit a bit of a wall, so I'm sharing it.

Try it live: https://dot-0.vercel.app/

Features:
• Real-time emotion engine with smooth interpolation
• Audio-reactive (voice input drives facial expressions)
• Swipe gestures to cycle through emotion presets
• Multiple face variants (minimal, digital, sketch, visionary styles)
• Video recording of avatar performances
• Memory bank for saving/restoring emotional states
• Export as PNG or animated GIF

Still very much a work in progress. Would love thoughts, feedback, or just to know if this resonates.
